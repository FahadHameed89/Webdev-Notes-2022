An inline style to adjust font size. Note that inline CSS should be used sparingly if you already have a .css file for styling
Block quotes can be used to make it known that a piece of text is a quotation
You can also make small text to make sentences smaller.
the em tag is used to emphasize text. Most browsers show this as italics.
Strong text is often rendered in bold.
Subscript makes letters appear smaller and lower than other text, often used in footnotes
Superscript is similar to subscript, but located above normal text.
Shows text in with a strikethrough.
Inserted text is often used with the del tag to show what was inserted along with the strikethrough text
